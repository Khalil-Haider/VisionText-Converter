VisionText Converter is an AI-powered tool that **extracts text from images and videos** in a good way. Users can turn visual content into **editable text** without manual typing.

---

## Key Features

- **Image & Video Support**: Works with `.png`, `.jpg`, `.jpeg`, `.mp4`, `.avi`, `.mov`, `.wmv`.
- **Video OCR**: Detects unique frames and extracts text with timestamps.
- **Export Options**: Save results as **PDF** or **DOCX**.
- **Session History**: Keeps track of processed files for easy access.
- **User-Friendly Interface**: Clean, interactive web UI built with Streamlit.

---

## How It Works

1. Select file type (Image/Video)  
2. Upload your file  
3. Processed text is displayed instantly  
4. Export or save as needed  

---

**In short:** This project turns visual content into editable text efficiently, making it perfect for content processing, data extraction, and automation tasks.
